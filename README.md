# cautious-lamp

Учебное портфолио фотографа (HTML + CSS + JS без сборки).

**Версия V2:** короткий `meta description`, плавная прокрутка по якорям с учётом липкой шапки и `prefers-reduced-motion`, кликабельные телефон и почта в контактах, видимый фокус (`:focus-visible`) у ссылок.

Контент правится в **`site-data.js`**. Разметка из данных строится в **`render.js`**, вставка в страницу — в **`app.js`**. Вёрстка — **`index.html`**, стили — **`styles.css`**.

## Запуск

1. Зависимости не нужны — это статические файлы.
2. Откройте `index.html` в браузере **или** запустите локальный сервер (так надёжнее для картинок и будущего размещения):

   ```bash
   # из папки проекта, если есть Python 3:
   python -m http.server 8080
   ```

   Затем в браузере: `http://localhost:8080`

3. Замените заглушки в `site-data.js` своими текстами, ценами и путями к фото.
